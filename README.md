# HRMS Frontend

## Project Overview

This is the frontend interface for the HRMS (Human Resource Management System) application. It provides a modern, responsive UI for managing employees and tracking attendance, built with React and Material-UI.

## Tech Stack

- React 18+
- Vite
- Material-UI
- Axios
- React Router

## Prerequisites

- Node.js 16 or higher

## Steps to Run the Project Locally

1. Install the required dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

The frontend will be running at http://localhost:5173 (or the port shown in the terminal)
